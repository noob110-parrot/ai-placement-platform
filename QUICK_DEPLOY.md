## 📦 Backend (Railway) - TO DO

### Quick Setup (5 minutes)

1. **Create Railway Project**
   - Go to https://railway.app
   - Sign in with GitHub
   - New Project → Deploy from GitHub repo
   - Select: `noob110-parrot/ai-placement-platform`

2. **Add Services**
   - Backend (auto-detected Python)
   - PostgreSQL (Add → Postgres)
   - Redis (Add → Redis)

3. **Add Environment Variables**
   ```
   DATABASE_URL=        (auto-generated by Railway Postgres)
   REDIS_URL=          (auto-generated by Railway Redis)
   SECRET_KEY=         your-secret
   OPENAI_API_KEY=     your-key
   ALGORITHM=HS256
   ACCESS_TOKEN_EXPIRE_MINUTES=30
   SMTP_HOST=smtp.gmail.com
   SMTP_PORT=587
   SMTP_USER=your-email
   SMTP_PASSWORD=your-app-password
   ENABLE_WHATSAPP_NOTIFICATIONS=true
   ENABLE_EMAIL_NOTIFICATIONS=true
   ENABLE_CALENDAR_SYNC=true
   ENABLE_AI_SUMMARIZER=true
   ```

4. **Deploy**
   - Click "Deploy" on backend service
   - Wait 2-3 minutes
   - Copy the public URL

5. **Update Vercel Frontend**
   - Go to https://vercel.com/dashboard
   - Select frontend project
   - Settings → Environment Variables
   - Update: `NEXT_PUBLIC_API_URL=https://your-backend-railway-url`
   - Redeploy (Deployments → 3-dots → Redeploy)

## 🛠️ Optional: PgAdmin & n8n on Railway

### PgAdmin (Database Manager)
1. In Railway project: Add → Container
2. Image: `dpage/pgadmin4:latest`
3. Env vars:
   ```
   PGADMIN_DEFAULT_EMAIL=admin@placement.edu
   PGADMIN_DEFAULT_PASSWORD=admin
   ```
4. Deploy
5. Access at public URL with admin/admin

### n8n (Automation)
1. In Railway project: Add → Container  
2. Image: `n8nio/n8n:latest`
3. Env vars:
   ```
   N8N_BASIC_AUTH_ACTIVE=true
   N8N_BASIC_AUTH_USER=admin
   N8N_BASIC_AUTH_PASSWORD=placement123
   N8N_PROTOCOL=https
   ```
4. Deploy
5. Access at public URL with admin/placement123
